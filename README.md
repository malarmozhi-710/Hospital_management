# Hospital_management
To design and implement a simple hospital management system that stores patient details in a CSV file and allows operations like adding, viewing, updating, and deleting patient records.

This project is a simple hospital management system that uses a CSV file to store patient data. The program performs basic operations such as adding new patients, viewing all records, updating patient details, and deleting patient records.
Initially, patient details are added to the CSV file. The program then reads and displays all records. It updates a specific patient’s details by modifying the data and rewriting the file. Similarly, it deletes a patient record by removing the corresponding row and saving the updated data back to the file.
This system demonstrates how file handling can be used to manage real-world data efficiently.
